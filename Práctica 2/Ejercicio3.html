Se pueden aplicar varias clases a la vez

En este caso tenemos tres reglas:

  • La primera se aplica a todos los elementos p que contengan la clase "quitar".
  • En la segunda el selector universal no realiza efecto alguno y la especificidad viene dada por la clase desarrollo
  • La tercera es similar a la anterior pero sin el selector universal, pero vimos que esto tiene el mismo efecto.

En la aplicación vemos como se aplica cada regla, aplicandose en diferentes elementos. También se muestra el caso de que haya dos clases en un mismo elemento. Aquí se aplican ambos estilos: el color rojo de "quitar" y el tamaño de "importante".
